# Bloom Today

**Because no mom should have to heal alone.**

Bloom Today is a postpartum emotional support platform that gives every new mother a personalized AI companion she can talk to, anytime, about anything, through real-time voice and video calls powered by the Gemini Live API.

## What makes it different

- **Full Personalization**: Name your companion, choose its voice, customize its personality, and give it instructions. It's your companion, on your terms.
- **Real-Time Voice & Video Calls**: Natural, conversational calls with a 3D animated avatar (Three.js + TalkingHead) that lip-syncs and reacts with contextual gestures.
- **Therapist Integration**: A mother can connect her therapist through a secure key. The therapist can send instructions to the AI companion, view a dedicated dashboard with Gemini-powered conversation analysis, and send notes.
- **Trusted Person System**: Connect a trusted person via a shareable key. They get their own dashboard in plain, everyday language with actionable suggestions generated by Gemini AI.
- **Intelligent Insights**: Every conversation is analyzed by Gemini to extract 8 signal scores (mood, energy, sleep, stress, support, self-kindness, coping, bonding), personalized tips, and YouTube resource recommendations.
- **Bloom Score & Streak**: A gamified engagement score that measures how consistently a mother seeks help, not her mental state. Miss a day and the streak breaks, gently encouraging consistent engagement.

## Prerequisites

Before you begin, make sure you have the following:

- **Node.js 20+** installed ([download](https://nodejs.org/))
- **npm** (comes with Node.js)
- **PostgreSQL database** (you can use [Neon](https://neon.tech/) for a free serverless Postgres instance)
- **Google Cloud project** with the following:
  - A **Google OAuth 2.0 Client ID** (for sign-in with Google)
  - A **Gemini API Key** from [Google AI Studio](https://aistudio.google.com/)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/veeradyani222/bloom.git
cd bloom
```

### 2. Set up the backend

```bash
cd backend
```

Create a `.env` file from the example:

```bash
cp .env.example .env
```

Fill in your `.env` values:

```env
PORT=8000
FRONTEND_ORIGIN=http://localhost:5000
DATABASE_URL=postgresql://USER:PASSWORD@HOST/DB?sslmode=require
GOOGLE_CLIENT_ID=your-google-oauth-web-client-id.apps.googleusercontent.com
APP_JWT_SECRET=replace-with-a-long-random-string
GEMINI_API_KEY=your-google-ai-studio-api-key
GEMINI_MODEL=gemini-2.5-pro
```

Install dependencies and run migrations:

```bash
npm install
npm run migrate
```

Start the backend server:

```bash
npm run dev
```

The backend will start on `http://localhost:8000`.

### 3. Set up the frontend

Open a **new terminal** and navigate to the frontend:

```bash
cd frontend
```

Create a `.env` file from the example:

```bash
cp .env.example .env
```

Fill in your `.env` values:

```env
VITE_API_BASE_URL=http://localhost:8000
VITE_GOOGLE_CLIENT_ID=your-google-oauth-web-client-id.apps.googleusercontent.com
VITE_GEMINI_API_KEY=your-gemini-api-key
VITE_GEMINI_LIVE_MODEL=gemini-2.5-flash-native-audio-latest
```

> **Note**: The `VITE_GOOGLE_CLIENT_ID` must match the `GOOGLE_CLIENT_ID` in your backend `.env`.

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

The frontend will start on `http://localhost:5000`.

### 4. Open the app

Navigate to **http://localhost:5000** in your browser. Sign in with Google to begin onboarding.

## Database

All tables are created automatically by the migration script (`npm run migrate`). The schema includes 13 migrations covering:

- `users` and `connections` (core user data and role-based connections)
- `companions` (AI companion configuration per user)
- `call_sessions` and `call_messages` (voice/video call transcripts)
- `call_session_analyses` (Gemini-powered conversation analysis)
- `therapist_notes` and `trusted_notes` (notes from therapist/trusted person to mom)
- `user_memories` (companion long-term memory from conversations)

## How It Works

1. **Onboarding**: Sign in with Google, enter your name, name your companion, choose a voice and avatar, optionally connect a therapist or trusted person.
2. **Voice/Video Calls**: Call your companion anytime. The Gemini Live API streams audio in real-time while a 3D avatar lip-syncs and reacts with contextual gestures.
3. **Post-Call Analysis**: After each call, Gemini 2.5 Pro analyzes the transcript and generates signal scores, themes, risk assessment, and personalized guidance for all three roles.
4. **Dashboard**: View your Bloom Score, mood/energy/sleep/calm rings, quick tips, and community notes from other moms.
5. **Multi-Role Access**: Therapists and trusted persons connect via unique 8-character keys and see their own tailored dashboards.
